Hilfe zum OMSI-Content-Tool

Willkommen in der lokalen Hilfe zum OMSI-Content-Tool.

Tastenkürzel

  • A - Alle Objekte markieren / demarkieren
  • Strg + D - Projektordner öffnen
  • Strg + E - Objekt Exportieren
  • Strg + F - Suchen
  • Strg + H - Objekt ein-/ ausblenden
  • Strg + I - Objekt Importieren
  • Strg + L - Logfile anzeigen
  • Strg + N - Neues Projekt erstellen
  • Strg + O - Projekt öffnen
  • Strg + P - Projekt packen
  • Strg + R - Projekt neu laden
  • Strg + S - Speichern
  • Strg + Shift + E - Projekteigenschaften
  • Strg + Shift + T - Text-Texturen Übersicht
  • Strg + Links - Zum vorherigen Wagenteil wechseln
  • Strg + Rechts - Zum nächsten Wagenteil wechseln
  • Entf - markiertes Objekt aus dem Projekt entfernen (die Datei wird nicht gelöscht)
  • F1 - Fahreransicht
  • F2 - Fahrgastansicht
  • F3 - Verfolgerkamera
  • F4 - Freie Kamera (Standard)
  • F5 - LOD-Filter
  • F6 - Objekte Fenster ein-/ ausblenden
  • F7 - Texturen Fenster ein-/ ausblenden
  • F8 - Eigenschaften Fenster ein-/ ausblenden

Consolenbefehle

Übersicht: omsi-content-tool.exe path [newTab] / [convert] (o3d, x, x3d, ...)

  • omsi-content-tool.exe
    		•  Startet das Tool im normalen Modus
  • path
    		•  Hier kann ein Pfad zu einer Projekt- oder 3D-Datei angegeben werden die direkt geöffnet werden soll
  • [newTab]
    		•  Öffnet das angegebene Projekt in einem neuen Tab. Kann nicht mit [convert] ... kombiniert werden. Angabe pfad erforderlich.
  • [convert] (format)
    		•  Konvertiert die angegebene O3D-Datei, Rückmeldung ausschließlich auf der Konsole. Kann nicht mit [newTab] kombiniert werden. Angabe pfad erforderlich.

    Beispiel 1: omsi-content-tool.exe "MAN_Lions_Coach_L.bus" newTab
    Beispiel 2: omsi-content-tool.exe "model\body.x" convert o3d
